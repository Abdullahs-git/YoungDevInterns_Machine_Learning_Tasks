# Task 2: Preprocess Data for Machine Learning

## Overview
This task focuses on preparing raw data for modeling using scikit-learn pipelines. It includes handling missing values, encoding categorical features, normalizing numerical values, and splitting the dataset for training and testing.

## Steps
1. Load the Titanic dataset.
2. Separate features and target.
3. Define numerical and categorical preprocessing pipelines.
4. Combine pipelines using ColumnTransformer.
5. Apply transformations and split into training and testing sets.

## Requirements
- Python 3.x
- pandas
- scikit-learn
