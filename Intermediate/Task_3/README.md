# Task 3: Create a Classification Report

## Overview
This task demonstrates how to generate a detailed classification report in scikit-learn. The report provides **precision**, **recall**, and **F1-score** for each class after training a classification model.

## Steps
1. Load the Wine dataset.
2. Split data into training and testing sets.
3. Scale features using StandardScaler.
4. Train a RandomForestClassifier.
5. Generate and display a classification report.

## Requirements
- Python 3.x
- scikit-learn
